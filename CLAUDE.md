# Yohaku - プロジェクトメモ

## サービス概要
- **コンセプト**: 日記 × 性格診断アプリ
- **キャッチコピー**: 「書くことで自分を知る」「日記を書くと、自分が見えてくる」
- **ターゲット**: 考えすぎてしまう人、自分を理解したい人
- **ターゲット地域**: 日本、タイ、ベトナム、フィリピン（韓国・台湾は対象外）

## ビジネス目標
- **売上目標**: 年間1,200万円（必達）
- **達成期限**: 3年以内
- **価格**: Premium 月額1,000円（年払いで2ヶ月分お得）
- **モデル**: フリーミアム（日記・基本診断は無料）

## 機能
- 毎日の日記からMBTIや心理傾向を分析・可視化
- パーソナリティカード（1日1枚）
- 心のトラッキング（外向性・内向性、思考・感情など）
- Premium: ダッシュボード、推移一覧、テーマ集計、公開設定

## デザイン方針
- **雰囲気**: カウンセリングっぽい、温かみ、落ち着き
- **NG**: ゲーミング風、派手なグラデーション
- **カラー**: stone / amber 系
- **フォント**:
  - 本文: Zen Maru Gothic（丸ゴシック）
  - 見出し: Shippori Mincho（明朝体）
  - ロゴ: EB Garamond（セリフ体）

## コピーライティングルール
- 「AI」という言葉は使わない
- 「一言」ではなく「日記」（しっかり書く印象）
- やさしく、押しつけがましくないトーン

## 技術スタック
- Next.js (App Router)
- next-intl（i18n）
- Tailwind CSS v4
- BlockNote（エディタ）
- next-themes

## 継続施策（アプリ内で実装）

### 診断結果の見せ方（フェーズ別）
- **初期（〜2週間）**: 毎日違う結果が出る「揺らぎ」を楽しませる
- **中期（1ヶ月〜）**: 結果が安定してきたことを示す（信頼度○%）
- **後期**: 「いつもと違う日」を際立たせる、過去との比較

### 文章力フィードバック（隠れた価値）
※LPでは訴求しない。使い続けるユーザーが発見する価値として実装
- 文章量の推移（今月は平均○字、先月比+○%）
- 表現の幅（感情を表す言葉が増えてきました）
- 継続日数バッジ（3日、7日、30日...）
- 語彙の広がり（今週使った新しい言葉: ○つ）
- 深さ（「なぜ」を問う文が増えています）

### 習慣化トリガー
- 夜のプッシュ通知（「今日はどんな一日でしたか？」）
- 書かなかった日も「お休みカード」として記録（罪悪感を減らす）
- 「今日の問いかけ」で書き出しのハードルを下げる
- 「1週間前のあなた」を振り返る通知
